# MediQueue — Tutor Booking System

A modern full-stack tutor booking web application built with Next.js, Express, and MongoDB.

## Live Site
Coming soon on Vercel

## Tech Stack
- **Frontend:** Next.js 16, Tailwind CSS v4, shadcn/ui, Framer Motion
- **Backend:** Node.js, Express.js, MongoDB
- **Auth:** Firebase + JWT
- **Deploy:** Vercel + Render

## Features
- Browse and search verified tutors
- Book sessions with real-time slot tracking
- Dark / Light theme toggle
- JWT-secured private routes
- Fully responsive modern UI
